# Kunnathur Carry Bee

This project is a fast, local, and friendly delivery service in Kunnathur. Users can order from their favorite local stores including groceries, medicines, food, and more.

## Tech Stack

*   **Frontend:** React, Vite, Tailwind CSS
*   **Backend:** Supabase
*   **Deployment:** Firebase

## Key Features

*   **Store Management:** Admins can manage stores, products, and orders.
*   **Real-time Order Tracking:** Users can track their orders in real-time.
*   **Push Notifications:** Users receive push notifications for order status updates.
*   **User Profiles:** Users can create profiles and manage their orders.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   npm
    ```sh
    npm install npm@latest -g
    ```

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username_/Project-Name.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the development server
    ```sh
    npm run dev
    ```
